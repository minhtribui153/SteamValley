# STEAM for Vietnam - CS 102 - STEAM Valley

| Keybind               	| Uses              	|
|-----------------------	|-------------------	|
| W \| Space \| UpArrow 	| Jump              	|
| A \| LeftArrow        	| Move Backward     	|
| D \| RightArrow       	| Move Forward      	|
| E                     	| Interact with NPC 	|
| F                     	| Shoot             	|
| Esc                   	| Pause game        	|
